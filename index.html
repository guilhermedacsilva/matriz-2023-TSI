A nova matriz de TSI

A nova matriz foi elaborada pensando no perfil do aluno que queremos formar. Remodelamos muitas disciplinas para que a matriz se aproxime do mercado de trabalho dos desenvolvedores de software. Resumo das alterações:

  • Quase metade da carga horária do curso será a distância.
  • O primeiro período se tornou nivelador. A ideia é que o aluno não tenha um grande choque quando ingressar na universidade.
  • A primeira disciplina de lógica de programação usará uma linguagem visual (quase não vai precisar escrever). Assim o aluno formará uma base de conhecimento para depois cursar uma disciplina de programação escrita.
  • O aluno aprenderá a utilizar o Linux desde o primeiro período.
  • O aluno aprenderá a utilizar o Git e Github desde o primeiro período. Hoje o Github faz parte do currículo do desenvolvedor web.
  • Maior quantidade de disciplinas de programação frontend.
  • Serão ensinadas basicamente duas linguagens de programação: Javascript e PHP.
  • Existirão disciplinas voltadas à aprendizagem de DevOps.
  • O aluno aprenderá a escrever testes automatizados para seus programas em todas as disciplinas de programação, criando assim uma cultura de testes.
  • Será ensinado inglês.
  • Será ensinada programação backend com e sem API.
  • Existirão duas disciplinas de programação para dispositivos móveis.
  • O aluno não precisará realizar atividades complementares.
  • Existirão duas disciplinas extensionistas.
  • Atualização das tecnologias utilizadas nas disciplinas.
  • Disciplinas removidas e diluídas durante o curso: matemática, interação humano-computador, design gráfico, gestão da qualidade e metodologia da pesquisa.
